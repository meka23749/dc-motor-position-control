# DC Motor Position Control — Model-Based Design

Full Model-Based Design (MBD) workflow for DC motor position control:
mathematical modeling, automatic PID tuning, C code generation with Embedded Coder, and SIL validation.

## Workflow
Mathematical Model → pidtune Auto-Tuning → Simulink → Embedded Coder → SIL Validation → STM32 ready
